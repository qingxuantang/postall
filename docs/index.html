🤖 AI Agent Integration

Built for AI agents. Use our CLI or integrate via skill files.

Agent CLI

JSON-first command line for AI agents. Works with OpenClaw, Claude, and more.

Terminal
$ pip install postall-agent
$ postall-agent generate --topic "AI trends" --platforms twitter,wechat,xhs 
{
  "success": true,
  "draft_id": "abc123",
  "contents": {
    "twitter": {"thread": ["Tweet 1...", "Tweet 2..."]},
    "wechat": {"title": "标题", "article": "文章..."},
    "xhs": {"title": "小红书", "content": "笔记..."}
  }
}
$ postall-agent publish --draft abc123
